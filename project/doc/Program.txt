Программе на вход поступает массив строк args с аргументами:
 - первый аргумент имя команды
 - последующе аргументы относятся уже к самой команде и их может быть разное количество в зависимости от выбраной команды
args[0] - первая команда, её значения проверяются с помощью конструкции switch(args[0]),
где уже зависимо от значения команды производится дальнейшая обработка команды, либо выход, с сообщением об ошибке.
Для дальнейшей обработки использовать функцию обёртку, которая распарсит аргументы в нужный формат (если это необходимо)
и передаст их уже основной фукнции. Основная функция проделав свою работу и возвращает результат. 
В функции main результат работы вспомогательных функций передается на вывод в консоль, т.е. для функции выглядеть это будет примерно так: Console.Write(sum(args));
